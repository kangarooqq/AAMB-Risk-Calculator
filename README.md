# AAMB低骨密度风险评估工具

一个基于多变量回归模型开发的在线计算器，用于估计亚裔绝经后女性发生低骨密度（BMD）的概率。

---

## 🔍 项目简介

**AAMB评分计算器（AAMB Score Calculator）** 是基于简约化多变量Logistic回归模型开发的临床辅助工具。

模型仅包含4个易于获取的临床变量：

- 年龄（Age）
- 初潮年龄（Age at menarche）
- 绝经年龄（Age at menopause）
- 体重指数（BMI）

用于快速估计个体发生低骨密度的概率。

本工具旨在：

- 辅助临床**风险分层**
- 支持**DXA检查转诊优先级决策**
- 适用于**基层医疗及资源有限环境**

## ⚙️ 模型公式

模型基于Logistic回归构建：
LP = 2.1186
+ 0.1591 × 年龄
− 0.1602 × 绝经年龄
+ 0.1760 × 初潮年龄
− 0.1646 × BMI


概率计算公式：
P = 1 / (1 + exp(−LP))


---

## 🩺 临床意义

本工具输出低骨密度的预测概率，并结合临床语境提供风险解释：

- **低风险倾向**  
  → 是否进行DXA检查需结合临床综合判断  

- **中等风险倾向**  
  → 如存在其他危险因素，可考虑进行DXA检查  

- **高风险倾向**  
  → 建议优先转诊进行DXA骨密度评估  

⚠️ 本模型主要用于**高风险人群识别（rule-in）**，而非普遍人群筛查或排除工具。
